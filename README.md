# Face Recognition Login System

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python  
- **Framework:** Django  
- **Libraries:** OpenCV, face_recognition, base64 (for decoding images), ContentFile (for image-to-data conversion)
